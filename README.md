# Example-HelloWorld

A 'Hello World' example using the STM32+NRF2401+. Built around the STM32 **BluePill DevBoard (STM32F103C8T6)**. Based on https://github.com/controllerstech/NRF24L01

## How to use with the BluePill:
+ Generate 2 project workspaces and initial code using STM32CubeIDE, using the pinout in the images below. One for TX, and another for RX.
+ Drop the files in their respective folders (Inc & Src). Replace as necessary.
+ Flash to STM32 through JLink
